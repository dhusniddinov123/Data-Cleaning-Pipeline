# 🧹 Automated Data Cleaning Pipeline

A robust Python-based pipeline designed to transform messy, "real-world" CSV data into a clean, structured, and analysis-ready format. This project automates common data-cleaning tasks using **Pandas**, ensuring data consistency and integrity.

---

## 🚀 Key Features
- **📂 Directory Management:** Automatically detects and creates the necessary `/data` directories if they are missing.
- **🔡 String Standardization:** 
    - Trims whitespace from names and genders (e.g., `" john "` becomes `"John"`).
    - Standardizes text to `Title Case` for names and `Capitalized` for gender.
- **🔢 Numeric Imputation:** 
    - Converts `age` and `salary` to numeric types.
    - Automatically fills missing values (`NaN`) with the column **mean** (average).
- **⚙️ Type Optimization:** Rounds ages and converts them to **integers** for cleaner reporting and better memory usage.
- **🛡️ Data Integrity:** 
    - Removes rows where the critical `name` field is missing.
    - Detects and removes duplicates based on a combination of `name` and `age`.
- **📊 Logging:** Provides a console summary report of the cleaning process (Initial vs. Final row counts).

## 📈 The "Before & After"
| Feature | Raw Data Example | Cleaned Data Example | Logic Applied |
| :--- | :--- | :--- | :--- |
| **Names** | `" john "` / `"JOHN"` | `"John"` | Strip + Title Case |
| **Gender** | `"male"` / `"MALE"` | `"Male"` | Strip + Capitalize |
| **Age** | `25.43` / `NaN` | `25` (Integer) | Round + Mean Impute |
| **Missing Data** | Empty rows | Removed | `dropna(subset=['name'])` |
